Getting Started

Spin up a node on the Gamecredits network.

Gamecore has Gamecredits built right in, so joining the peer-to-peer network is simple.

1

Install gamecore with NPM

You'll need to have Node.js and ZeroMQ dependencies installed, and be running as a user without root privileges. For more detailed info, check out our expanded guide.

npm install -g gamecore
2

Start your node

Once you install gamecore, the gamecored command will be available in your terminal. Run it to start your node. 

gamecored
3

Open Insight

Your gamecore node is now connecting to the peer-to-peer network and downloading the blockchain.

You can watch the download progress by opening Insight, the bundled blockchain explorer, in a web browser.

open http://localhost:3001/insight/
